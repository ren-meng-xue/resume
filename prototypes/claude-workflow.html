// 03 · 自动化守卫
Hooks
三层钩子覆盖写入前 / 写入后 / 收尾三个时机，自动完成格式化、QA、权限放行。
触发时机 匹配 脚本 行为
PostToolUse Write | Edit quality-post-write.sh
每次写完自动格式化
Python → ruff format + ruff check --fix
TS/JS/CSS/JSON → prettier --write
Stop 全局 stop-test-gate.sh
收尾前强制质量门控，最多阻 3 次
ruff check lint 通过
pytest backend/tests/unit/ 通过
tsc --noEmit 类型检查通过
PermissionRequest 只读工具 permission-auto-readonly.sh
Read / Glob / Grep / git status / git diff 等自动放行，不打断工作流
非只读操作走正常权限弹窗
PreToolUse Write video_to_chinese_pre_write.py
video-to-chinese skill 写入前预处理
配合 PostToolUse 完成视频字幕翻译闭环
// 04 · 并行子代理
Agents
单窗口内调度多个专职 subagent，后端 + 前端并行实现，reviewer + tester 独立验收。
backend-engineer
claude-sonnet-4-6 · Read Write Edit Bash Glob Grep
按 plan.md 实现 API、数据库变更、业务逻辑，编写单元测试，修复 reviewer / tester 反馈。
frontend-engineer
claude-sonnet-4-6 · Read Write Edit Bash Glob Grep
按 plan.md 实现页面、组件、状态管理，编写前端测试，修复 reviewer / tester 反馈。
reviewer
claude-sonnet-4-6 · Read Glob Grep Write
检查实现是否符合 plan、代码质量、安全边界，写入 review.md，输出 APPROVED / CHANGES_REQUESTED。
tester
claude-sonnet-4-6 · Read Bash Glob Grep
执行测试套件、验收标准检查、回归验证，输出 PASSED / FAILED 结论。
